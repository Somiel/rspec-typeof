# RspecTypeof

Welcome to rspec_typeof, with this gem you can use "typeof" expectation to your tests


## Installation

Add this line to your application's Gemfile in the group :development, :test:

```ruby
gem 'rspec_typeof'
```

And then execute:

    $ bundle

## Usage


```ruby
expect(true).to typeof(:string_or_nil_or_true)
```
